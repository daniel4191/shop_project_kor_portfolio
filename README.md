# 📚 DanielDB: 프리미엄 디지털 콘텐츠 마켓플레이스

> **"단순한 구매를 넘어, 최상의 디지털 경험을 제공하는 고해상도 콘텐츠 플랫폼"**
> 
> **DanielDB**는 고해상도 이미지와 전자책을 최적화된 환경에서 유통하고 관리하는 **Django 기반의 전문 커머스 플랫폼**입니다. 무손실 압축 기술과 고품질 렌더링 뷰어를 통해 창작자의 가치를 온전히 독자에게 전달합니다.

### 🔗 라이브 데모: [https://danieldb.work](https://danieldb.work)

---

## 📋 프로젝트 개요

**DanielDB**는 디지털 자산의 품질 저하 없이 사용자에게 전달하기 위한 기술적 고민에서 시작되었습니다. 기존 마켓플레이스들의 일률적인 압축 방식을 탈피하여, **이미지는 무손실 최적화**를, **전자책은 고해상도 렌더링**을 제공함으로써 상업적 활용도가 높은 고품질 마켓을 지향합니다.

---

## 🚀 핵심 기능 (6대 핵심 역량)

### 📖 Part 1. 고해상도 전자책 경험 (E-Book Viewer)

#### 1️⃣ 고해상도 무손실 미리보기
* **"구매 전, 텍스트 한 글자까지 선명하게 확인하세요."**
* PyMuPDF를 활용한 **300 DPI 고해상도 렌더링** 시스템 구축.
* 확대 시에도 글자가 깨지지 않는 2배 확대(Matrix) 렌더링 기법을 적용하여 실제 종이책과 유사한 가독성 제공.

#### 2️⃣ 스마트 전용 뷰어 & 관리
* **"구매한 즉시 읽고, 내 서재에서 언제든 꺼내보세요."**
* 별도의 툴 없이 브라우저에서 즉시 구동되는 전용 PDF 뷰어 탑재.
* 하이라이트 및 필기 등 독서 몰입도를 높여주는 직관적인 사용자 경험 제공.

---

### 🎨 Part 2. 상업용 이미지 자산 (Image Assets)

#### 3️⃣ 무손실 이미지 최적화 시스템
* **"용량은 가볍게, 품질은 원본 그대로."**
* Pillow 기반의 지능형 리사이징 및 무손실 압축 알고리즘 적용.
* 창작자가 의도한 색감과 디테일을 보존하면서도 웹 로딩 속도를 최적화.

#### 4️⃣ 상업적 활용을 위한 다운로드 시스템
* **"상업적 용도로도 안심하고 계속해서 사용하세요."**
* 구매한 이미지는 '내 갤러리'를 통해 기간 제한 없이 무제한 반복 다운로드 가능.
* 라이선스 관리가 용이한 소장용 시스템으로 비즈니스 활용성 극대화.

---

### 💳 Part 3. 신뢰할 수 있는 이커머스 (E-Commerce)

#### 5️⃣ 원스톱 결제 및 이용 프로세스
* **"결제 즉시 홈으로 돌아갈 필요 없이 콘텐츠로 바로 연결됩니다."**
* **Toss Payments** 연동을 통한 안전하고 빠른 결제 환경 제공.
* 결제 성공 시 구매한 상품 유형(전자책/이미지)에 최적화된 **'즉시 이용 버튼'** 자동 생성.

#### 6️⃣ 스마트 주문 취소 및 권한 관리
* **"투명한 정책으로 고객의 신뢰를 확보합니다."**
* 디지털 콘텐츠의 특성을 고려한 **'열람 전 7일 이내 취소'** 자동화 로직 구현.
* 이미 사용한 콘텐츠에 대해서는 철저한 권한 관리를 통해 비즈니스 손실 방지.

## 🛠 기술 스택

| 분류 | 사용 기술 |
| :--- | :--- |
| **Backend** | Python 3.13, Django 5.2 |
| **Frontend** | HTML5, CSS3, JavaScript, Bootstrap 5 |
| **Infra** | AWS Lightsail, Nginx, Gunicorn |
| **Libraries** | PyMuPDF (PDF 렌더링), Pillow (이미지 최적화) |
